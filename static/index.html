📖 Help & Docs

rc is a browser-based remote terminal. Each tab runs a command on the server. Here's how to get the most out of it.

Tab Management

🟢 Running — process is active
🟡 Idle — no output for a while (pulsing)
🔴 Exited — process has ended; click ↻ Restart at the bottom to relaunch
⚫ Disconnected — remote agent went offline
🟣 Purple ring — tab is from a remote agent

Switch tabs with Alt+1 – Alt+9. Drag tabs to reorder.

Split Pane

Hover over any tab to see the ⧉ split icon. Click it to send that tab to the right pane for side-by-side monitoring. Use the ✕ button on the split header to unsplit. On mobile, split pane appears as a slide-out drawer.

Tab Menu ☰

The menu button at the right end of the tab bar provides:

✕ Close disconnected tabs — remove offline agent tabs
↺ Reset tab order — restore original order
🔄 Check for Updates — update rc from the UI
ℹ About & Licenses — version and credits

File Upload

When the server is started with --upload, an upload icon appears in the header. Click it to upload a file to the server's working directory. Files are not overwritten — if a file already exists, rename it first.

If no upload icon is visible, the server was started without --upload.

Mobile & Touch

On touch devices, use the floating keyboard button (bottom-right) for arrow keys, Ctrl+C, Tab, Esc, and clipboard paste.
